## 2026-02-01 - Committed Environment Secrets
**Vulnerability:** The `.env` file containing environment variables (potentially secrets) was tracked in the git repository.
**Learning:** Initial project setup or template generation failed to include `.env` in `.gitignore`, leading to accidental tracking of local configuration.
**Prevention:** Always verify `.gitignore` includes sensitive file patterns (`.env`, `*.pem`, etc.) before the first commit. Use pre-commit hooks to scan for secrets.
## 2026-03-25 - [ChartStyle XSS & CSS Injection via dangerouslySetInnerHTML]\n**Vulnerability:** The ChartStyle component used React's dangerouslySetInnerHTML to inject CSS variables into a <style> tag without sanitizing the id, key, and color props, allowing potential XSS and CSS injection breakouts.\n**Learning:** When using dangerouslySetInnerHTML, even within <style> tags, props must be sanitized. Structural characters like <, >, ;, }, [, ], ", and ' can be used to break out of CSS property boundaries or close the style tag prematurely.\n**Prevention:** Sanitize all dynamic props interpolated into dangerouslySetInnerHTML by stripping out structural characters using a regex like /[<>;}\[\]"']/g before rendering.
